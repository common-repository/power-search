This plugins allows to search content from post, comment, pages in WordPress admin side.

== Description ==
* This plugins allows to search content from post, comment, pages in WordPress admin side.
* It is helpful when you have large number of posts or pages or comments and you want to search specific word from the content. 
* This plugins contain filter which allows to search content from posts or pages or comments or all.

== Installation ==
1. Upload the full directory into your wp-content/plugins directory
2. Activate the plugin at the plugin administration page
